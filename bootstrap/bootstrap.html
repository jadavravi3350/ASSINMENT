(4). What do you know about the Bootstrap Grid System?

== > Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive. Below is an example and an in-depth look at how the grid comes together. New to or unfamiliar with flexbox?

  1. Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content.
  2. It's built with flexbox and is fully responsive.

  3. Below is an example and an in-depth look at how the grid comes together.

  4. Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page.
  5. The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

  6. The Bootstrap 5 grid system has six classes:

  7. .col- (extra small devices - screen width less than 576px)
  8. .col-sm- (small devices - screen width equal to or greater than 576px)
  9. .col-md- (medium devices - screen width equal to or greater than 768px)
  10. .col-lg- (large devices - screen width equal to or greater than 992px)
  11. .col-xl- (xlarge devices - screen width equal to or greater than 1200px)
  12. .col-xxl- (xxlarge devices - screen width equal to or greater than 1400px)

(5). What is the difference between Bootstrap 4 and Bootstrap 5

== > Bootstrap 5 has six levels: xs, sm, md, lg, xl, and xxl. Bootstrap 4 includes jquery and all associated plug-ins. Jquery has been deleted and replaced with vanilla JS and several functional plug-ins in Bootstrap 5. Bootstrap 4 provides a few colors for a web page.

(8). In Bootstrap 4, what is flexbox?

== > Flexbox. The biggest difference between Bootstrap 3 and Bootstrap 4 is that Bootstrap 4 now uses flexbox, instead of floats, to handle the layout. The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.
